# Installation
In your terminal: 

1. Clone repository

"git clone https://github.com/MatiasTorterolo/notepad-api.git"

2. Install dependencies

If use Maven:
"mvn clean install"

If use Gradle:
"./gradlew build"

# Use
In your terminal:

If use Maven:
"mvn spring-boot:run"

If use Gradle:
"./gradlew bootRun"
